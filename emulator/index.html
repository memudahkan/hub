PlayStation

Platform gaming legendaris Sony dengan segudang game ikonik yang masih seru dimainkan sampai sekarang.

PS1

  • DuckStation — Emulator PS1 kaya fitur untuk mobile dan PC. Play Store / GitHub
  • ePSXe — Emulator PS1 klasik, lebih ramah untuk device lawas. Play Store
  • FPse — Emulator PS1 karya developer asal Perancis, alternatif ePSXe. Play Store / FPseNG (FPse64)

PS2

  • PCSX2 🖥️ — Emulator PS2 untuk PC. GitHub
  • NetherSX2 — Emulator PS2 untuk Android, penerus AetherSX2. GitHub
  • ARMSX2 🌱 — Emulator PS2 untuk device berbasis ARM, termasuk Android. Play Store / GitHub
  • Play! 🌱 — Emulator PS2 untuk mobile dan PC, aktif dikembangkan. Play Store

PSP

  • PPSSPP — Emulator PSP untuk Android dan PC. Play Store / GitHub

PS3

  • RPCS3 — Emulator PS3 open-source untuk Windows, Linux, macOS, dan FreeBSD. GitHub
  • aPS3e 🌱 — Emulator PS3 native untuk Android, masih early-stage. Play Store / GitHub
  • RPCSX UI Android 🌱 — Emulator PS3 eksperimental untuk Android, early-stage juga.

PS Vita

  • Vita3K 🌱 — Emulator PS Vita open-source untuk Android dan PC, eksperimental. GitHub / Rilis Android

Nintendo

Identik dengan game yang fun, inovatif, dan punya karakter populer macam Mario dan Zelda.

NES

  • NES.emu — Emulator NES berbayar di Play Store, tapi gratis di GitHub. Play Store / GitHub 🆓

Game Boy/Game Boy Color

  • Pizza Boy C — Emulator Android dengan fokus pada skin, tampilan visual, dan nuansa handheld yang lebih modern.
  • GBC.emu — Alternatif yang lebih minimalis dan konsisten dengan keluarga EX untuk Game Boy dan Game Boy Color. Play Store / GitHub 🆓

SNES

  • Snes9x EX+ — Cocok buat yang ingin emulator SNES klasik dengan kontrol responsif dan tampilan yang simpel. Play Store / GitHub

N64

  • M64Plus FZ — Salah satu opsi emulator Nintendo 64 untuk Android yang paling dikenal, terutama kalau butuh pengaturan yang lebih lengkap. Play Store

Game Boy Advance

  • Pizza Boy A — Emulator Game Boy Advance yang polished, fleksibel, dan enak dikustomisasi.
  • GBA.emu — Pilihan yang lebih simpel dengan pendekatan klasik dan satu keluarga dengan emulator EX lain. Play Store / GitHub 🆓

GameCube/Wii

  • Dolphin — Emulator dual-purpose, bisa mainin GameCube dan Wii. Play Store / GitHub

Nintendo DS

  • melonDS — Opsi emulator Nintendo DS yang cukup jelas jalur proyeknya dan nyaman dipakai. GitHub / Android Port / Play Store 🔀

Nintendo 3DS

  • Azahar — Proyek penerus Citra yang masih relevan untuk emulasi Nintendo 3DS modern. Play Store / GitHub

Wii U

  • Cemu — Emulator Wii U PC yang paling bisa diandalkan. GitHub / Android fork 🌱

Sega

Rumah bagi game klasik dengan nuansa arcade yang kuat dan evergreen.

Mega Drive / Master System

  • MD.emu — Emulator Sega Genesis/Mega Drive, Sega CD, dan Master System, dari developer yang sama dengan NES.emu. Play Store / GitHub 🆓

Saturn

  • Yaba Sanshiro 2 — Emulator Sega Saturn untuk Android. Play Store / GitHub

Dreamcast

  • Flycast — Emulator Dreamcast, Naomi, Naomi 2, dan Atomiswave gratis untuk Android dan PC. Play Store
  • redream — Emulator Dreamcast multi-platform, beberapa fitur dibatasi, bisa dibuka dengan IAP. Play Store

Mobile Classics

Emulator untuk Symbian, N-Gage, Java/J2ME, dan Mophun di Android maupun PC.

Symbian

  • EKA2L1 — Emulator Symbian OS dan N-Gage untuk Android dan PC. Play Store / GitHub

Java/J2ME

  • J2ME Loader — Emulator game Java untuk Android dengan dukungan banyak game 2D dan sebagian game 3D. Play Store
  • FreeJ2ME-Plus 🖥️ — Emulator Java/J2ME open-source untuk PC.
  • KEmulator nnmod 🖥️ — Emulator J2ME cross-platform, berbasis KEmulator 1.0.3. GitHub

Mophun

  • Nofun — Emulator Mophun untuk Android dan PC, dibuat dengan Unity. GitHub

Multi-system

Emulator all-in-one, bisa memainkan banyak sistem sekaligus.

  • RetroArch — Aplikasi yang menggabungkan berbagai emulator (core) dengan berbagai fitur mumpuni. Play Store ⌛ / F-Droid
  • Lemuroid — Emulator multi-system sederhana untuk Android. Play Store / GitHub
  • Mesen 🖥️ — Emulator multi-system untuk PC, mendukung NES, SNES, Game Boy, GBA, PC Engine, SMS/Game Gear, dan WonderSwan. GitHub

PC Emulation

Tool untuk menjalankan aplikasi atau game PC di Android, mulai dari compatibility layer sampai platform gaming hybrid.

  • Winlator — Aplikasi Android untuk menjalankan aplikasi dan game Windows lewat Wine dan Box86/Box64. GitHub
  • GameHub — Platform gaming Android dari GameSir yang menggabungkan PC emulation, cloud gaming, dan streaming dalam satu aplikasi. Play Store
  • GameNative — Frontend eksperimental untuk menjalankan game PC milikmu di Android dengan pendekatan yang lebih praktis. GitHub
  • MiceWine — Proyek open-source untuk menjalankan aplikasi dan game Windows di Android dengan basis Wine dan Box64.

Frontend

Peluncur koleksi game/emulator dengan tampilan yang kece badai, bikin smartphone Android serasa console gaming.

  • Daijisho — Launcher game retro gratis, cocok buat yang baru mulai. Play Store
  • Beacon — Launcher minimalis untuk Android yang fokus pada koleksi game klasik.
  • ES-DE — Frontend multi-platform yang fleksibel dengan nuansa console-style yang lebih serius. Patreon
