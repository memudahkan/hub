PlayStation

Platform gaming legendaris Sony dengan segudang game ikonik yang masih seru dimainkan sampai sekarang.

PS1

  • DuckStation — Emulator PS1 kaya fitur untuk mobile dan desktop. Play Store / GitHub
  • ePSXe — Emulator PS1 klasik, lebih ramah untuk device lawas. Play Store

PS2

  • NetherSX2 — Emulator PS2 penerus AetherSX2.
  • PCSX2 — Emulator PS2 untuk desktop. GitHub

PSP

  • PPSSPP — Emulator PSP untuk Android dan desktop. Play Store / GitHub

PS3

  • RPCS3 — Emulator PS3 open-source untuk Windows, Linux, macOS, dan FreeBSD. GitHub
  • 🌱 aPS3e — Emulator PS3 native untuk Android, masih early-stage. Play Store / GitHub
  • 🌱 RPCSX UI Android — Emulator PS3 eksperimental untuk Android, early-stage juga.

PS Vita

  • 🌱 Vita3K — Emulator PS Vita open-source untuk Android dan desktop, eksperimental. GitHub / Rilis Android

Nintendo

Identik dengan game yang fun, inovatif, dan punya karakter populer macam Mario dan Zelda.

NES

  • NES.emu — Emulator NES berbayar di Play Store, tapi gratis di GitHub. Play Store / 🆓 GitHub

SNES

  • Snes9x EX+ — Cocok buat yang ingin emulator SNES klasik dengan kontrol responsif dan tampilan yang tidak ribet. Play Store / GitHub

Game Boy / Game Boy Color

  • Pizza Boy C — Emulator Android dengan fokus pada skin, tampilan visual, dan nuansa handheld yang lebih modern.
  • GBC.emu — Alternatif yang lebih minimalis dan konsisten dengan keluarga EX untuk Game Boy dan Game Boy Color. Play Store / 🆓 GitHub

Game Boy Advance

  • Pizza Boy A — Cocok untuk pengguna yang suka emulator polished, fleksibel, dan enak dikustomisasi.
  • GBA.emu — Pilihan yang lebih simpel dengan pendekatan klasik dan satu keluarga dengan emulator EX lain. Play Store / 🆓 GitHub

N64

  • M64Plus FZ — Salah satu opsi Android yang paling dikenal untuk Nintendo 64, terutama kalau butuh pengaturan yang lebih lengkap. Play Store

Nintendo DS

  • melonDS — Opsi DS yang cukup jelas jalur proyeknya dan nyaman dipakai. GitHub / Android Port / 🔀 Play Store

Nintendo 3DS

  • Azahar — Proyek penerus Citra yang masih relevan untuk emulasi 3DS modern. Play Store / GitHub

Wii U

  • Cemu — Emulator Wii U desktop yang paling bisa diandalkan. GitHub / 🌱 Android fork

Sega

Rumah bagi game klasik dengan nuansa arcade yang kuat dan evergreen.

Mega Drive / Master System

  • MD.emu — Emulator Sega Genesis/Mega Drive, Sega CD, dan Master System, dari developer yang sama dengan NES.emu. Play Store / 🆓 GitHub

Saturn

  • Yaba Sanshiro 2 — Emulator Sega Saturn untuk Android. Play Store / GitHub

Dreamcast

  • Flycast — Emulator Dreamcast, Naomi, Naomi 2, dan Atomiswave untuk Android dan desktop. Play Store

Mobile Classics

Emulator untuk Symbian, N-Gage, dan Java/J2ME di Android maupun desktop.

Symbian

  • EKA2L1 — Emulator Symbian OS dan N-Gage untuk Android dan desktop. Play Store / GitHub

Java/J2ME

  • J2ME Loader — Emulator game Java untuk Android dengan dukungan banyak game 2D dan sebagian game 3D. Play Store
  • FreeJ2ME-Plus — Emulator Java/J2ME open-source untuk desktop.
  • KEmulator nnmod — J2ME emulator cross-platform, berbasis KEmulator 1.0.3. GitHub

Multi-system

Emulator all-in-one, bisa memainkan banyak sistem sekaligus.

  • RetroArch — Aplikasi yang menggabungkan berbagai emulator (core) dengan berbagai fitur mumpuni. ⌛ Play Store / F-Droid
  • Lemuroid — Emulator multi-system sederhana untuk Android. Play Store / GitHub

PC Emulation

Tool untuk menjalankan aplikasi atau game PC di Android, mulai dari compatibility layer sampai platform gaming hybrid.

  • Winlator — Menjalankan aplikasi dan game Windows di Android lewat Wine dan Box86/Box64. GitHub
  • GameHub — Platform gaming Android dari GameSir yang menggabungkan PC emulation, cloud gaming, dan streaming dalam satu aplikasi. Play Store
  • GameNative — Frontend eksperimental untuk menjalankan game PC milikmu di Android dengan pendekatan yang lebih praktis. GitHub
  • MiceWine — Proyek open-source untuk menjalankan aplikasi dan game Windows di Android dengan basis Wine dan Box64.

Frontend

Peluncur koleksi game/emulator dengan tampilan yang kece badai.

  • Daijisho — Launcher game retro gratis, cocok buat yang baru mulai. Play Store
  • Beacon — Launcher minimalis untuk Android yang fokus pada koleksi game klasik tanpa antarmuka yang terlalu ramai.
  • ES-DE — Frontend multi-platform yang fleksibel dengan nuansa console-style yang lebih serius. Patreon

Guide

Panduan singkat untuk membantu memilih atau menyiapkan emulator tertentu.

  • Pilih emulator sesuai platform — Gunakan Android untuk akses cepat, desktop untuk kompatibilitas lebih luas.
  • Catatan BIOS dan file tambahan — Tempatkan catatan legal dan teknis di sini.
